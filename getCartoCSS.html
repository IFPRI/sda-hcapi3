Generate CartoCSS snippet to symbolize raster tiles

Usage

getCartoCSS(var, pal = "BuGn", legend = TRUE, ...)

Arguments

var
HarvestChoice variable name
pal
optional Brewer color palette used for plotting
legend
if TRUE returns HarvestChoice legend, otherwise returns default legend
...
any argument passed to getLayer(), e.g. by=FS_2012, iso3=GHA
iso3
optional country or region filter (3-letter code)

Value

character, CartoCSS snippet for requested variable

Description

Generate CartoCSS snippet to symbolize raster tiles
