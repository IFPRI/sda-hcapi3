Summarize HarvestChoice 5-arc-minute layers over WKT geometri(es)

Usage

getLayerWKT(var, wkt, ...)

Arguments

var
character array of variable names (all types are accepted)
wkt
WKT representation of a spatial object (points, multipoints, or polygons)
...
other arguments passed to getLayer (e.g. by, iso3, collapse)

Value

a data.table of var indicators summarized across wkt geometry

Description

This function intersects any spatial object (WKT point, multipoints, or polygons) with HarvestChoice 5-arc-minute grid and returns a list of intersected gridcell IDs (CELL5M codes) to pass along as argument to getLayer.
