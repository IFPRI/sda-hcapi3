Convert CELL5M layers to raster and/or tabular formats for download

Usage

genFile(var, iso3 = "SSA", by = NULL, format = c("csv", "geojson", "tif", "dta", "asc", "rda", "grd"), ...)

Arguments

var
character array of variable codes, passed to getLayer
iso3
character array of ISO3 country or region codes, passed to getLayer
by
character array of variable codes to summarize by, passed to getLayer
format
output format c("csv", "json", "tif", "dta", "asc", "grd", "rda")
...
any other optional argument getLayer, e.g. by, collapse.} } { character, array of generated file names } { Package any result from getLayer into the user-specified tabular or raster format. Also includes a README file with metadata and citation details. Currently supported export formats include CSV (csv), STATA (dta), GeoJSON (json), GeoTIFF (tif), R raster (grd), R Data (rda), and ESRI ASCII raster (asc).
