Generate HarvestChoice data use terms, citation, and variable metadata

Usage

genReadme(var)

Arguments

var
character array of CELL5M variable names

Value

character, path to generated README file

Description

Generate HarvestChoice data use terms, citation, and variable metadata

Examples

# README for soil organic carbon concentration
f <- genReadme("sod_d5")
print(readLines(f))
