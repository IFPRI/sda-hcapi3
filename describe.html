Return HarvestChoice variable metadata

Usage

describe(var, cat, version, raster = FALSE, as.class = "data.table", css = "json")

Arguments

var
optional character array of HarvestChoice variable codes
cat
optional category filter (fuzzy matches)
version
optional version filter
raster
return only raster variables
as.class
simple "data.table" or if "list" group variables by category
css
include Carto CSS rules

Value

a data.table or hierarchical list of variable metadata

Description

Return HarvestChoice variable metadata

Examples

# Show complete metadata for all variables under 'Cassava' category
describe(cat="cassava", raster=T)
