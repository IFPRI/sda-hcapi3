Return HarvestChoice variable metadata

Usage

describe(var, cat, version, raster = FALSE, by.category = FALSE, css = "json")

Arguments

var
character array of HarvestChoice variables
cat
optional category filter
version
optional version filter
raster
return only raster variables
css
include Carto CSS rules
by.group
group variables by category

Value

a data.table of variable metadata

Description

Return HarvestChoice variable metadata

Examples

# Show complete metadata for all variables under 'Cassava' category
describe(cat="cassava", raster=T, by.category=T)
