Plot HarvestChoice 5-arc-minute spatial indicators

Usage

genPlot(var, iso3 = "SSA", pal = character(0), layout = "default", style = NULL, ...)

Arguments

var
character array of variable codes to plot
iso3
optional ISO3 country or region code(s)
pal
optional Brewer color palette used for plotting, e.g. "Blues"
layout
one of "default" or "thumbnail" to control legend and axes
style
passed to classIntervals
...
any argument passed to classIntervals, e.g. n=9 and dataPrecision=0.

Value

raster plot

Description

Method to plot HarvestChoice rasters with mutiple layout and symbology options. See examples below. Note that calling genPlot(...) is equivalent to calling the convenience function hcapi(..., format="plot").

Details

API call: generate 2 plots showing farming systems and 2012 population density for Ghana

GET all generated plots in a ZIP archive

$ wget http://hcapi.harvestchoice.org/ocpu/tmp/x03d5aa8e98/graphics/last/png?width=640 $ wget http://hcapi.harvestchoice.org/ocpu/tmp/x03d5aa8e98/graphics/last/svg

Examples

# Generate standard raster plot of 2012 population density for sub-Saharan Africa
genPlot("PD12_TOT", pal="YlOrRd", style="pretty", n=8)
